# Api_to_hash script

Simple script to convert an string into a hash.

## How to run it:

  1. Ensure you got python 3.7 installed in your computer
  2. Download the repository and unzip it
  3. cd to the repository folder
  4. Open command line into the folder
  5. Write: python api_to_hash.py
  6. Introduce your PUBLIC API KEY
  7. Copy the hashcode
  8. Add it to the form
